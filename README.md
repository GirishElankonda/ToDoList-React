# MultiPage React Router Application

A modern, responsive multi-page React application built with Vite, React Router, Tailwind CSS, and React Icons.

## 🚀 Features

- **Multi-Page Navigation**: Smooth client-side routing using `react-router-dom` (Home, About, Products, and Contact pages).
- **Responsive Navigation Bar**: A clean, modern header with hover animations and responsive layouts.
- **Tailwind CSS Styling**: Slick, utility-first styling for a premium feel.
- **Dynamic Icons**: Integration of high-quality icons using `react-icons`.

## 📁 Project Structure

```text
vite-project/
├── public/                  # Static assets (favicons, icons)
├── routes/                  # Page route components
│   ├── About.jsx
│   ├── Contact.jsx
│   ├── Home.jsx
│   └── Products.jsx
├── src/
│   ├── App.jsx              # Main routing configuration
│   ├── Navbar.jsx           # Global Navigation component
│   ├── main.jsx             # React entry point
│   └── index.css            # Stylesheets and Tailwind imports
├── index.html               # Main HTML document
├── package.json             # NPM dependencies and scripts
└── vite.config.js           # Vite configuration
```

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/GirishElankonda/MultiPage-React.git
   cd MultiPage-React
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
